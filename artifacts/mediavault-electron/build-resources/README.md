# build-resources

Place your app icons here before packaging:

| File | Format | Notes |
|------|--------|-------|
| `icon.png` | 512×512 PNG | Linux (AppImage, .deb, .rpm) |
| `icon.ico` | Multi-size ICO | Windows |
| `icon.icns` | ICNS bundle | macOS |

The `icon.png` included here is a placeholder. Replace it with your final
512×512 artwork before building production installers.

See `BUILDING.md` for commands to generate `.ico` and `.icns` from a PNG.
